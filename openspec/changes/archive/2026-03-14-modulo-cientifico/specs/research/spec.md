# Research Specification

## Purpose
Este dominio define el comportamiento del Agente Investigador para la recuperación de fuentes académicas, validación de metadatos y generación de síntesis científica con rigor académico.

## Requirements

### Requirement: Academic Source Retrieval
El sistema DEBE ser capaz de consultar bases de datos académicas (Semantic Scholar/OpenAlex) para obtener artículos relevantes basados en una consulta del usuario.

#### Scenario: Successful search for academic papers
- GIVEN a valid search query (e.g., "Impact of AI on medical diagnosis").
- WHEN the user initiates the research search.
- THEN the system MUST return a list of at least 5 relevant academic articles.
- AND each article MUST include metadata: Title, Authors, Year, Abstract, and DOI.

### Requirement: Grounded Synthesis (RAG)
El sistema DEBE generar una síntesis de la información recuperada utilizando únicamente los datos de los artículos encontrados para evitar alucinaciones.

#### Scenario: Synthesis with valid citations
- GIVEN a set of academic articles with validated metadata.
- WHEN the AI agent generates the synthesis.
- THEN it MUST include in-text citations in APA 7 format (e.g., Smith, 2023).
- AND every cited article MUST exist in the initial set of retrieved articles.

### Requirement: DOI Validation
El sistema DEBE verificar que cada cita generada por la IA corresponda a un DOI real y presente en los metadatos recuperados.

#### Scenario: Validation of generated citations
- GIVEN the synthesis generated by the AI.
- WHEN the system post-processes the text.
- THEN it MUST cross-reference each citation against the list of DOIs retrieved from the API.
- AND if a citation cannot be validated, it MUST be removed or marked for correction.

### Requirement: Reference List Generation
El sistema DEBE generar una sección de referencias al final del documento con el formato APA 7ma Edición.

#### Scenario: Generation of reference list with hanging indent
- GIVEN a document with multiple in-text citations.
- WHEN the user requests the generation of the reference list.
- THEN the system MUST list all cited sources in alphabetical order.
- AND each entry MUST follow the APA 7 format.
- AND each entry MUST have a hanging indent of 1.5em (or equivalent CSS).
